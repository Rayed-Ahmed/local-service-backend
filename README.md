# 🛠️ Local Service Marketplace – Backend

A scalable **NestJS backend API** for a **Local Service Marketplace** where customers can find and book nearby service providers such as electricians, plumbers, cleaners, and more.

The system supports **three types of users**:

- 👤 **Customers** – browse services and create bookings  
- 🧰 **Service Providers** – manage services and bookings  
- 🛡️ **Admins** – manage platform operations

The backend is built with **NestJS**, **PostgreSQL**, and **JWT authentication**, following modular and scalable architecture.

---

# 🚀 Features

### 👤 Customer
- Register and login
- Browse service providers
- Book services
- View booking history
- Upload profile image

### 🧰 Service Provider
- Register and login
- Manage services
- Accept or reject bookings
- View assigned bookings
- Upload service images

### 🛡️ Admin
- Manage customers
- Manage service providers
- Monitor bookings

# 🏗️ Tech Stack

| Technology | Description |
|---|---|
| **NestJS** | Backend framework |
| **PostgreSQL** | Database |
| **TypeORM** | ORM for database interaction |
| **JWT** | Authentication |
| **Nodemailer** | Email sending |
| **ConfigModule** | Environment configuration |
